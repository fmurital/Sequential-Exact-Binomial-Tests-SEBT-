SEBT addresses the post hoc identification problem by treating the set of streams as multiple hypotheses and applying formal multiple testing procedures. We consider four approaches: raw p values (no adjustment), Bonferroni correction, Holm correction, and the Benjamini Hochberg (BH) procedure for false discovery rate control.
The Bonferroni and Holm procedures control the family‑wise error rate (FWER), whereas the BH procedure controls the false discovery rate (FDR) (Bonferroni, 1936; Holm, 1979; Benjamini & Hochberg, 1995). To evaluate these procedures, we conducted an extensive Monte Carlo simulation mirroring the data-generating process in Muritala et. al, 2026 (https://arxiv.org/pdf/2604.12095), using the following parameters: the number of streams k=10, the in control proportion p_0=0.5, the smoothing parameters λ and control limit multipliers L that achieve ARL_0 of 370 and 500, and the shift magnitudes δ=0,0.05,0.10,0.15,0.2,0.3,0.4 which are identical to those used in Muritala et. al, 2026.

We evaluate each method’s sensitivity, specificity, false discovery rate, family wise error rate, and Type I error. The results demonstrate that raw p-values lead to inflated error rates, whereas the Benjamini-Hochberg procedure offers an optimal balance between error control and detection power. This work extends Brown (2019) by embedding formal, error-controlled multiple-testing procedures, Bonferroni, Holm, and Benjamini-Hochberg, into the post-signal diagnostic framework, providing the first rigorous multiplicity-adjusted identification method for a nonparametric MSP chart. To our knowledge, this provides the first rigorous multiplicity‑adjusted identification method for a nonparametric MSP chart.
